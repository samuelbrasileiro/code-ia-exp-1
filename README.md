# Exam Builder (React + Node + JSON)

Simple monorepo for creating multiple-choice exams, generating randomized PDFs, exporting answer keys, and correcting student answers via CSV.

## Project Structure

- `apps/frontend` - React + Vite UI
- `apps/backend` - Express API + PDF/CSV generation
- `packages/shared` - Shared TypeScript types
- `data` - JSON persistence (created automatically, gitignored)

## Prerequisites

- Node.js 18+
- npm 9+

## Install

```bash
npm install
```

## Run (Dev)

In two terminals:

```bash
npm run dev:backend
```

```bash
npm run dev:frontend
```

- Backend: `http://localhost:3000`
- Frontend: `http://localhost:5173`

## Build (Production)

```bash
npm run build
```

### Backend

```bash
npm -w apps/backend run build
npm -w apps/backend run start
```

### Frontend

```bash
npm -w apps/frontend run build
npm -w apps/frontend run preview
```

## CSV Formats

Answer key CSV (generated by the app):

```
examId,variantId,questionId,correctChoiceId
...
```

Student answers CSV (uploaded for correction):

```
studentId,questionId,selectedChoiceId
...
```
